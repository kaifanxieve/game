# HOI4 Country Ideas Database

Extracted and compiled from Hearts of Iron IV, covering all country ideas in both Chinese and English versions with console command prefix.

## Files

- all_ideas_prefixed_CN.txt — Chinese version with idea ID, Chinese name, Chinese effect description, prefixed with add_ideas
- all_ideas_prefixed_EN.txt — English version with idea ID and raw effect keys, prefixed with add_ideas
- all_ideas_translated.txt  — Full Chinese translated version (no prefix)
- all_ideas_complete.txt    — Full English raw version (no prefix)

## Usage

Each line follows this format:

    add_ideas IDEA_ID

Copy and paste any line directly into the HOI4 console to apply the idea.

## Data Sources

- Game data: Hearts of Iron IV, copyright Paradox Interactive
- Chinese localisation: Official Simplified Chinese localisation files
- Data compilation and translation mapping: Xiao Fan

## Coverage

- 71 countries
- 3915 country ideas
- All DLC included (SEA, WUW, AAT, BBA, NSB, GOE, etc.)
# HOI4 国家精神数据库

整理自《钢铁雄心4》全部国家精神数据，提供中英文双版本及控制台指令格式。

## 文件说明

- all_ideas_prefixed_CN.txt — 中文版，含精神ID、中文名称、中文效果描述，每行带 add_ideas 前缀
- all_ideas_prefixed_EN.txt — 英文版，含精神ID、原始效果key，每行带 add_ideas 前缀
- all_ideas_translated.txt  — 中文翻译完整版（无前缀）
- all_ideas_complete.txt    — 英文原始完整版（无前缀）

## 使用方法

每行格式如下：

    add_ideas 精神ID

直接复制精神ID行粘贴进游戏控制台即可生效。

## 数据来源

- 游戏数据：Hearts of Iron IV，版权归 Paradox Interactive 所有
- 中文本地化：游戏官方简体中文本地化文件
- 数据整理与翻译对照：小烦

## 覆盖范围

- 71个国家
- 3915条国家精神
- 包含所有DLC内容（SEA、WUW、AAT、BBA、NSB、GOE等）
